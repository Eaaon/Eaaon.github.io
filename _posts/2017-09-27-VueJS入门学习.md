### 一、入门代码

Vue.js 的核心是一个允许采用简洁的模板语法来声明式的将数据渲染进 DOM 的系统：

1、导入CSS和JS

```html
<script src="https://unpkg.com/vue"></script>
```
2、Html代码
```html
   <div id="demo">
       <p>{{message}}</p>
       <input v-model="message">
   </div>
```
3、JS代码
```js
var demo =new Vue({
          el:'#demo',
          data:{
              message:'hello vue!'
          }

      })
```
